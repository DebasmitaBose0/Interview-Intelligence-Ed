# AI Interview Platform

An interactive web application designed to help users prepare for job interviews through AI-powered mock interviews, coding test environments, and deeply insightful analytics reports.

## Structure

- **`client/`**: Frontend codebase developed with React, modern CSS, and Vite.
- **`server/`**: Express backend API layer routing user details, sessions, and generating analytical report entities.
- **`ai-engine/`**: Housing logic components for resume/job description parsing and local LLM/Ollama integrations.
- **`database/`**: Schema migrations and queries.

## Getting Started

1. Install dependencies across environments.
2. Configure the `.env` setup file.
3. Spin up local database and servers.
